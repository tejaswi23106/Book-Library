# 📚 Book Management API

A full backend REST API built using Node.js, Express, and MongoDB.

---

## 🚀 Features

- ✅ User Signup  
- ✅ User Login (JWT Authentication)  
- ✅ Password Hashing (bcryptjs)  
- ✅ Protected Routes  
- ✅ Add Book  
- ✅ Get All Books  
- ✅ Get Book by ID  

---
